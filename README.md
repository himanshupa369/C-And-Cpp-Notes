# C-And-Cpp-Notes

| Stage         | Tool Used    | Input File | Output File      | Description                   |
| ------------- | ------------ | ---------- | ---------------- | ----------------------------- |
| Preprocessing | Preprocessor | `.c`       | `.i`             | Handles directives and macros |
| Compilation   | Compiler     | `.i`       | `.s`             | Converts to assembly          |
| Assembly      | Assembler    | `.s`       | `.o`             | Converts to machine code      |
| Linking       | Linker       | `.o`, libs | `.exe` / `a.out` | Produces final executable     |
